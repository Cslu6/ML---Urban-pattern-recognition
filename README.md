# ML Projet Kaggle

Projet privé pour collaboration à deux sur un dataset Kaggle.

## Structure (suggestion)
- `data/` : données locales (non versionnées)
- `notebooks/` : notebooks d’exploration
- `src/` : code source
- `models/` : modèles entraînés (non versionnés)
- `outputs/` : sorties et prédictions (non versionnées)

## Setup
1. Créer/activer l’environnement conda
2. Installer les dépendances

## Collaboration
- Dépôt GitHub privé
- Inviter le collaborateur via Settings → Collaborators
